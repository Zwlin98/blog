## Ubuntu 及各种常见 Linux 发行版软件源

**这里大力推荐清华的一键 tuna 脚本**
一键使用 TUNA 脚本，又名 oh-my-tuna。它目前支持更改下列软件的配置，将其直接指向 TUNA 镜像站：
+ Anaconda
+ Arch Linux
+ TeX Live (by tlmgr)
+ Debian GNU/Linux
+ Homebrew
+ PyPI
+ Ubuntu Linux
详细使用参考其 [github](https://github.com/tuna/oh-my-tuna)
